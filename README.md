# 💪 My Gym Tracker

Una web app completa per tracciare i tuoi allenamenti in palestra, con tutti i dati salvati localmente nel browser.

## ✨ Funzionalità

### 🏋️ Gestione Allenamenti
- Crea e traccia allenamenti completi
- Aggiungi esercizi con serie, ripetizioni e pesi
- Visualizza lo storico degli allenamenti
- Aggiungi note personali per ogni allenamento

### 💪 Database Esercizi
- Crea il tuo database personale di esercizi
- Organizza per categoria (Petto, Schiena, Gambe, ecc.)
- Visualizza quante volte hai eseguito ogni esercizio
- Aggiungi note tecniche per ogni esercizio

### 📊 Statistiche e Progressi
- Grafico allenamenti mensili
- Top 5 esercizi più frequenti
- Grafico progressi per esercizio (carico massimo nel tempo)
- Statistiche settimanali

### 💾 Salvataggio Locale
- Tutti i dati salvati in localStorage
- Nessun server necessario
- Privacy totale: i tuoi dati rimangono sul tuo dispositivo
- Funziona offline

## 🚀 Come Usare

1. **Apri `index.html` nel browser**
   - Basta fare doppio click sul file o aprirlo con il browser preferito

2. **Crea i tuoi esercizi**
   - Vai alla tab "Esercizi"
   - Clicca "Nuovo Esercizio"
   - Compila nome, categoria e note

3. **Registra un allenamento**
   - Vai alla tab "Allenamenti"
   - Clicca "Nuovo Allenamento"
   - Seleziona gli esercizi e inserisci serie/ripetizioni/pesi
   - Salva!

4. **Monitora i progressi**
   - Vai alla tab "Statistiche"
   - Visualizza grafici e tendenze
   - Seleziona un esercizio per vedere i progressi di carico
